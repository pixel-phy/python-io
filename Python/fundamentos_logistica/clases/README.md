# Clases

## Día 1: Clases, Atributos y Métodos de Instancia
**Conceptos:** `class`, `__init__`, `self`, atributos y métodos de instancia.  
**Aplicación IO:** Modelado de componentes de red (Nodos, Aristas) y variables de decisión.  
**Ejercicios:** VariableBinaria, RestriccionLineal, Arista, SistemaCola, PlanProduccion.

## Día 2: Métodos Especiales de Representación
**Conceptos:** `__str__`, `__repr__`, `__format__`.  
**Aplicación IO:** Visualización limpia de estados de modelos, rutas óptimas y soluciones de sistemas.  
**Ejercicios:** Representaciones mejoradas para VariableBinaria, SistemaCola, RedFlujo, PlanProduccion y sistema integrado de gestión de rutas.

## Día 3: Atributos de Clase y Métodos de Clase
**Conceptos:** Atributos de clase, `@classmethod`, constructores alternativos.  
**Aplicación IO:** Variables globales del modelo y creación de objetos desde CSV/strings.  
**Ejercicios:** Parámetros globales en colas, constructores alternativos para nodos, configuración global de redes, fábrica de variables y gestor de modelos de optimización.

## Día 4: Métodos Estáticos y Encapsulamiento
**Conceptos:** `@staticmethod`, atributos privados, properties, getters y setters.  
**Aplicación IO:** Funciones matemáticas auxiliares y protección de parámetros críticos.  
**Ejercicios:** Calculadora de distancias, variable con encapsulamiento, sistema de inventario, red de flujo protegida y sistema de optimización de producción.

## Día 5: Relaciones entre Clases
**Conceptos:** Composición, agregación, asociación.  
**Aplicación IO:** Construcción de modelos complejos a partir de componentes.  
**Ejercicios:** Modelo PL con restricciones, modelo PL con variables, simulación de colas, problema de asignación y optimización de cartera.

## Día 6: Ejercicios Integradores de Repaso
**Conceptos:** Optimización, refactorización y buenas prácticas (PEP 8).  
**Aplicación IO:** Mejora de código existente y aplicación de estándares de calidad.

## Día 7: Proyecto Integrador
**Resultado Final:** Sistema completo de gestión de red de transporte operativa, integrando todos los conceptos aprendidos durante la semana.
